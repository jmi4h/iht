# IHT

**Film Productions & Industrial Motion**

Official website and digital archive for IHT.

[View the Site](https://jmi4h.github.io/iht/)

---

### Phase 1: Establish the Platform

The current phase of this project is to establish the core visual identity of IHT and deploy a stable, minimalist web interface. This acts as the central hub for all future media output.

**Milestones:**
* Design and production of the official `Resolve 01` motion ident.
* Development of a functional, single-page video gallery.
* Successful deployment on GitHub Pages with integrated favicon (`blackv.png`).

### Production Workflow

This project is built using a direct, high-performance tech stack, focusing on load times and clean user experience.

| Component | Standard |
| :--- | :--- |
| **Markup** | Standard HTML5 |
| **Styling** | Clean CSS3 (Industrial Minimalist Theme) |
| **Hosting** | GitHub Pages (CDN Acceleration) |
| **Video** | Embedded YouTube Player (1080p Standard) |
